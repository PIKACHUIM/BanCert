### 国产流氓、娱乐软件和不受欢迎的软件屏蔽工具

> 众所周知，现如今的国产软件流氓推广几乎无处不在，让人防不胜防

> 你要没有火眼金睛或者稍有疏忽，各种全家桶绝对会让你抓狂到吐血

> 那么有没有什么一劳永逸的方法，禁止电脑安装这些流氓软件呢？

# 使用限制

1. Vista以上UAC开启

2. 软件要有数字证书

# 使用方法

## 禁用软件

双击【一键屏蔽证书.bat】

## 恢复证书

双击【一键洗白证书.ps1】

### 如果无法恢复

双击【系统证书管理.lnk】
